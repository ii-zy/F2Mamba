# F2Mamba
Official implementation of F2Mamba for Image Manipulation Localization(IML).

## 🔥 Highlights
F2Mamba is a fine-grained image manipulation localization framework designed to localize tampered regions with high precision. The code and pretrained models are provided to reproduce the results reported in our paper.

Pretrained weights are available here: [Google Drive](https://drive.google.com/drive/folders/18m70d44A6YRSUZ09FTbTzHstMQ3U9wWi?usp=share_link)
